Software I Use

I use Linux as my primary operating system. The flavour of Linux that I use is ArchLinux. Which is great on multiple counts. One, it follows a minimalist approach of giving you only the bare minimum packages and configurations by default. Two, it is a rolling release which means that you get the updates for all the packages you have installed, as soon as they are published. The third reason is the presence of the AUR (Arch User Repository) which enables you to install a lot of niche packages with just a single command. All the complicated stuff like the download and configuration are taken care of for you under-the-hood.

Desktop Environment

I do not use any of the full fledged desktop environments (DE) like Gnome or KDE. Rather I use i3 which is just a tiling window manager. Until recently I was using Openbox which was a floating window manager. Using just a window manager means that you get to pick and choose and build your own desktop environment. My desktop environment includes the following apps.

  • polybar as my status bar
  • dunst for my notifications
  • xcompmgr for window effects
  • dmenu as my application switcher
  • sxhkd for keyboard shortcuts

Common Applications that I use

Low-level configurations

Web services
