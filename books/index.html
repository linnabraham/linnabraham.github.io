Machine Learning Resources

These are books I have read (at least partly) and know to be good.

  • Deep Learning with Python by Francois Chollet
  • Machine Learning: An Algorithmic Perspective by Stephen Marsland
  • Statistics, Data Mining, and Machine Learning in Astronomy by Željko Ivezić, Andrew J. Connolly , Jacob T. VanderPlas & Alexander Gray
  • Deep Learning by Ian Goodfellow , Yoshua Bengio
  • Pattern Recognition and Machine Learning by Christopher M. Bishop

These are books I believe are great and which I would like to read.

  • Information Theory, Inference and Learning Algorithms by David J. C. MacKay
  • An Introduction to Statistical Learning by Gareth James, Daniela Witten, Trevor Hastie, Robert Tibshirani & Jonathan Taylor
